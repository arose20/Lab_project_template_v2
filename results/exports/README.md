# Exports

This directory contains datasets prepared for external use or interoperability with other tools.

## Examples

- AnnData (.h5ad)
- Seurat (.rds)
- Loom
- Matrix Market files
- CSV exports
- Cellxgene-ready datasets

## Rules

- Exports should not be the canonical source of data.
- They should be reproducible from processed datasets.
- Include version information where appropriate.
